# Contributing

Danke für dein Interesse an diesem Projekt! Hier sind Richtlinien für Beiträge.

## Wie kann ich helfen?

### Fehler melden
- Öffne ein Issue mit Label `bug`
- Beschreibe das Problem so detailliert wie möglich
- Füge Screenshots bei, falls relevant

### Features vorschlagen
- Öffne ein Issue mit Label `enhancement`
- Beschreibe das gewünschte Feature
- Erkläre warum es nützlich wäre

### Code beitragen
1. Fork erstellen
2. Feature-Branch anlegen (`git checkout -b feature/mein-feature`)
3. Änderungen committen
4. Pull Request erstellen

## Coding Standards

- TypeScript strict mode
- Vue 3 Composition API mit `<script setup>`
- Vanilla CSS (keine Frameworks)
- ESLint + Prettier Konfiguration befolgen

## Reifegrad beachten

Dieses Projekt folgt einem Reifegrad-Modell. Bitte beachte:

- **Grad 1 Features** haben Priorität
- Größere Features für Grad 2+ bitte erst nach Absprache
- Breaking Changes vermeiden

## Fragen?

Öffne ein Discussion für allgemeine Fragen.
